# Sorting-algorithm-visualizer
A terminal tool that visualizes how sorting algorithms work

<br>
<img src="image.png" alt="display image" width="500"/>

# Usage
<br>
Example command:
<br>

```
java sortVisualizer.java --speed 10 -b
#10 miliseconds per frame and bubble sort
```

`java sortVisualizer.java <option>`
<br>
`--speed <int in milisec>` speed of the animation
<br>
`-b` bubble sort
<br>
`-q` quick sort
<br>
`-s` selection sort
<br>
`-m` merge sort
